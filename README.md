# Mobile-Automation-devy

**How to Start**

**Install**

1. Node JS

2. Java SDK & JAVA_HOME

3. Android Studio & ANDROID_HOME

4. Appium Desktop
   
	Install package.json with command

		npm install


**How to run**

1. Connect to your device android or IOS
   
3. Setup desired capabilities in file wdio.conf.js
   
5. Run test
 
 		npm run wdio
